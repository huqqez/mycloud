# 🎧 MyCloud — менеджер музыки на Python

Консольное приложение для управления музыкальной коллекцией: медиатека, плейлисты, поиск песен и загрузка текстов из интернета. Данные сохраняются между запусками.

## ✨ Возможности
- 📋 Медиатека песен (название + исполнитель)
- ➕ Добавление новых песен
- 🔍 Умный поиск по названию или исполнителю
- 📂 Плейлисты — свои подборки песен
- 🌐 Загрузка текстов песен из интернета (API lyrics.ovh)
- 💾 Сохранение данных в JSON — программа помнит песни после закрытия

## 🚀 Как запустить
1. Установить Python 3
2. Установить библиотеку: `pip install requests`
3. Запустить: `python mycloud.py`

## 🛠️ Что использовано
- Классы и объекты (ООП)
- Списки, словари, кортежи, циклы, условия
- Работа с файлами и форматом JSON
- Запросы в интернет (`requests` + API)
- Обработка ошибок (`try / except`), type hints

## 📌 Примечание
Стартовый список песен — демонстрационный, просто для примера при первом запуске. Программа позволяет добавлять свои песни, и они сохраняются в `library.json`.
